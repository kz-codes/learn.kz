# Overview of Apache Spark

## What is apache spark?

It is a **Unified Computing Engine** and set of **Libraries** for **Parallel data Processing on Computing Clusters**

- What is Unified?
  - wide range of tasks over one computing engine
- Computing Engine?
  - Limited to Computing
  - Doesn't store data
  - Data Sources - HDFS, IDBC/ODBC,Azure etc
  - Works with almost all data sources
- Libraries
  - Set of Prewritten Code -> Modules and Classes
- Paralell data processing?
  - multiple Executors do independent work that accumulated into a result
- Computing Clusters?
  - Used for Parallel Computing
  - A Master Computer servers taskes to a Slave Computers (Master - Slave)
